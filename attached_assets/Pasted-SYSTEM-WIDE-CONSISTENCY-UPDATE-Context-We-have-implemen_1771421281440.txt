SYSTEM-WIDE CONSISTENCY UPDATE

Context:
We have implemented major structural and wording refinements in the Structured Financial Brief (report view).

These same refinements MUST now be replicated across the entire application UI to ensure:

- Logical consistency
- Legal defensibility
- Terminology alignment
- No conflicting interpretations
- No advisory language in UI while report is compliant

The modelling engine remains unchanged.
This task affects presentation, wording, structure, and visibility only.

=================================================

“Should qualify”
“May qualify”
“May not qualify”

“Within selected benchmark thresholds”

- Report labels

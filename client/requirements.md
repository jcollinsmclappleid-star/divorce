## Packages
recharts | Financial charts and projections
zustand | Global state management for the financial engine
framer-motion | Smooth transitions and animations
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes

## Notes
The application uses a client-side "Engine" for calculations to ensure privacy and responsiveness.
State is persisted to localStorage via Zustand, with optional server-side saving.
Disclaimer banner must be visible on all pages.
